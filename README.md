# nappy hugo theme

A starter theme for [Hugo](https://gohugo.io).

## Getting Started

From the root of your Hugo site, clone the theme into `themes/nappy` by running:

````
git submodule add https://github.com/njncalub/nappy-hugo-theme.git themes/nappy
````
